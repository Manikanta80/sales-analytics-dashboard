# Retail Sales Analytics Dashboard

This project is a Sales Analytics Dashboard built using Python, Flask, MySQL and Chart.js.

## Features
- Total Revenue KPI
- Total Profit KPI
- Top Performing Stores
- Revenue Visualization

## Technologies
- Python
- Flask
- MySQL
- HTML
- CSS
- Chart.js

## Run Project

python app.py

Open:
http://127.0.0.1:5000
