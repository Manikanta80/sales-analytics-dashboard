# 📊 Retail Sales Analytics Dashboard

A **Sales Analytics Dashboard** built using **Python, Flask, MySQL, HTML, CSS and Chart.js** to analyze retail store performance and visualize important business metrics.

This project demonstrates how raw sales data can be transformed into **meaningful insights using data analytics and visualization**.

---

# 🚀 Project Overview

Retail businesses generate a lot of sales data every day.
This dashboard helps analyze that data and answer questions like:

• What is the **total revenue generated**?
• How much **profit** was earned?
• Which **stores are performing the best**?
• How does **store revenue compare visually**?

The dashboard fetches data from a **MySQL database** and displays analytics in an interactive interface.

---

# ✨ Features

✔ 💰 **Total Revenue KPI** – Displays overall revenue from all sales
✔ 📈 **Total Profit KPI** – Calculates profit based on product cost and selling price
✔ 🏬 **Top Performing Stores** – Shows the stores generating the highest revenue
✔ 📊 **Revenue Visualization** – Charts for comparing store performance
✔ 🔄 **Dynamic Data Loading** – Data is fetched from backend APIs
